Describes the part of the user interface represented by this widget.

The framework calls this method in a number of different situations. For example:

  • After calling initState.
  • After calling didUpdateWidget.
  • After receiving a call to setState.
  • After a dependency of this State object changes (e.g., an InheritedWidget referenced by the previous build changes).
  • After calling deactivate and then reinserting the State object into the tree at another location.

This method can potentially be called in every frame and should not have any side effects beyond building a widget.

The framework replaces the subtree below this widget with the widget returned by this method, either by updating the existing subtree or by removing the subtree and inflating a new subtree, depending on whether the widget returned by this method can update the root of the existing subtree, as determined by calling Widget.canUpdate.

Typically implementations return a newly created constellation of widgets that are configured with information from this widget's constructor, the given BuildContext, and the internal state of this State object.

The given BuildContext contains information about the location in the tree at which this widget is being built. For example, the context provides the set of inherited widgets for this location in the tree. The BuildContext argument is always the same as the context property of this State object and will remain the same for the lifetime of this object. The BuildContext argument is provided redundantly here so that this method matches the signature for a WidgetBuilder.

Design discussion

Why is the build method on State, and not StatefulWidget?

Putting a Widget build(BuildContext context) method on State rather than putting a Widget build(BuildContext context, State state) method on StatefulWidget gives developers more flexibility when subclassing StatefulWidget.

For example, AnimatedWidget is a subclass of StatefulWidget that introduces an abstract Widget build(BuildContext context) method for its subclasses to implement. If StatefulWidget already had a build method that took a State argument, AnimatedWidget would be forced to provide its State object to subclasses even though its State object is an internal implementation detail of AnimatedWidget.

Conceptually, StatelessWidget could also be implemented as a subclass of StatefulWidget in a similar manner. If the build method were on StatefulWidget rather than State, that would not be possible anymore.

Putting the build function on State rather than StatefulWidget also helps avoid a category of bugs related to closures implicitly capturing this. If you defined a closure in a build function on a StatefulWidget, that closure would implicitly capture this, which is the current widget instance, and would have the (immutable) fields of that instance in scope:

class MyButton extends StatefulWidget {
  ...
  final Color color;

  @override
  Widget build(BuildContext context, MyButtonState state) {
    ... () { print("color: $color"); } ...
  }
}

For example, suppose the parent builds MyButton with color being blue, the $color in the print function refers to blue, as expected. Now, suppose the parent rebuilds MyButton with green. The closure created by the first build still implicitly refers to the original widget and the $color still prints blue even through the widget has been updated to green.

In contrast, with the build function on the State object, closures created during build implicitly capture the State instance instead of the widget instance:

class MyButtonState extends State<MyButton> {
  ...
  @override
  Widget build(BuildContext context) {
    ... () { print("color: ${widget.color}"); } ...
  }
}

Now when the parent rebuilds MyButton with green, the closure created by the first build still refers to State object, which is preserved across rebuilds, but the framework has updated that State object's widget property to refer to the new MyButton instance and ${widget.color} prints green, as expected.

See also:

  • StatefulWidget, which contains the discussion on performance considerations.

Implementation

@override
Widget build(BuildContext context) {
  // get padding necessary to guarantee safe padding
  final double safePaddingTop = MediaQuery.of(context).padding.top;
  final double safePaddingBottom = MediaQuery.of(context).padding.bottom;

  TaskScreenCubit taskScreenCubit;
  if (widget.poi.poiData.task.taskType == TaskType.DETECTIVE) {
    // startState is dependent on whether message has been received from unity.
    taskScreenCubit = DetectiveScreenCubit(poi: widget.poi, startState: currentState);
  } else if (widget.poi.poiData.task.taskType == TaskType.GTS) {
    // startState is dependent on whether message has been received from unity.
    taskScreenCubit = GuessTheSizeScreenCubit(poi: widget.poi, startState: currentState);
  } else {
    taskScreenCubit = NoARTaskScreenCubit(poi: widget.poi, startState: TaskScreenFinishedNoModal(answer: "", score: 0, taskData: null));
  }
  // Start unity scene correctly if unitywidget already is created.
  return BlocProvider<ArScreenCubit>(
    create: (BuildContext context) => ArScreenCubit(),
    child: BlocBuilder<ArScreenCubit, ArScreenState>(builder: (context, state) {
      final ArScreenCubit arScreenCubit = BlocProvider.of<ArScreenCubit>(context);
      arScreenCubit.taskScreenState = TaskScreenAnswerNotSelected(progress: 0);
      return BlocProvider<TaskScreenCubit>(
        create: (BuildContext context) => taskScreenCubit,
        child: BlocBuilder<TaskScreenCubit, TaskScreenState>(
          builder: (context, state) {
            return Scaffold(
              body: Stack(
                key: _stackKey,
                children: [
                  // Unity screen widget. This is first, because it should be under all below widgets on the screen
                  if (hasAR)
                    UnityWidget(
                      onUnityCreated: onUnityCreated,
                      isARScene: true,
                      safeMode: true,
                      onUnityMessage: onUnityMessage,
                    ),

                  if (hasAR && !downloaded) Center(key: _loadKey, child: CircularProgressIndicator()),

                  // AR task info screen
                  Visibility(
                    // only visible when the task is ongoing and type is Detective or GuessTheSize and the model is placed.
                    visible: (!(taskScreenCubit.state is TaskScreenFinished) &&
                        modelPlaced &&
                        (widget.poi.poiData.task.taskType == TaskType.DETECTIVE || widget.poi.poiData.task.taskType == TaskType.GTS)),
                    child: Container(
                      margin: EdgeInsets.fromLTRB(c4, max(safePaddingTop + 5, c4), c4, 0),
                      child: ArScreenInfo(
                        taskType: widget.poi.poiData.task.taskType,
                        imageUrl: (widget.poi.poiData.task.taskType == TaskType.DETECTIVE)
                            ? (widget.poi.poiData.task.taskData as DetectiveTaskData).questionImage
                            : null,
                        question: (widget.poi.poiData.task.taskType == TaskType.DETECTIVE)
                            ? (widget.poi.poiData.task.taskData as DetectiveTaskData).question
                            : (widget.poi.poiData.task.taskType == TaskType.GTS)
                                ? (widget.poi.poiData.task.taskData as GuessTheSizeTaskData).question
                                : "",
                      ),
                    ),
                  ),

                  // AR Slider for Guess The Size task
                  Visibility(
                    // only visible when the task is ongoing and type is GuessTheSize and the model is placed.
                    visible: (!(taskScreenCubit.state is TaskScreenFinished) && widget.poi.poiData.task.taskType == TaskType.GTS && modelPlaced),
                    child: Column(
                      mainAxisAlignment: MainAxisAlignment.end,
                      children: [
                        Padding(
                          padding: EdgeInsets.fromLTRB(0, 0, 0, 0),
                          child: Text(
                            "Guess the size",
                            style: CubeFonts.cubeSansSerif(color: cubeWhite, fontSize: 20),
                          ),
                        ),
                        Padding(
                          padding: EdgeInsets.fromLTRB(0, 0, 0, 2 * c8 + safePaddingBottom),
                          child: Row(
                            mainAxisAlignment: MainAxisAlignment.center,
                            children: [
                              Padding(
                                padding: EdgeInsets.fromLTRB(c2, 0, 0, 0),
                                child: Image.asset(
                                  "assets/png/mouse.png",
                                  width: c6,
                                ),
                              ),
                              Container(
                                width: MediaQuery.of(context).size.width - 3 * c8,
                                child: Slider(
                                  activeColor: cubeGreen,
                                  inactiveColor: cubeDarkGreen,
                                  onChanged: (value) {
                                    // only update if state is not finished
                                    if (!(taskScreenCubit.state is TaskScreenFinished))
                                      currentState = TaskScreenAnswerSelected(answer: exp(value).toString(), progress: 0);
                                    setState(() {
                                      sliderValue = value;
                                    });
                                    SetSize((exp(value) / 100).toString());
                                  },
                                  value: ((widget.poi.poiData.task.taskType == TaskType.GTS))
                                      ? (modelPlaced)
                                          ? sliderValue
                                          : log((widget.poi.poiData.task.taskData as GuessTheSizeTaskData).start)
                                      : 1,
                                  min: (widget.poi.poiData.task.taskType == TaskType.GTS)
                                      ? log((widget.poi.poiData.task.taskData as GuessTheSizeTaskData).min)
                                      : 0,
                                  max: (widget.poi.poiData.task.taskType == TaskType.GTS)
                                      ? log((widget.poi.poiData.task.taskData as GuessTheSizeTaskData).max)
                                      : 2,
                                ),
                              ),
                              Padding(
                                padding: EdgeInsets.fromLTRB(0, 0, c2, 0),
                                child: Image.asset(
                                  "assets/png/elephant.png",
                                  width: c6,
                                ),
                              ),
                            ],
                          ),
                        ),
                      ],
                    ),
                  ),

                  // Go to Map Screen (x)
                  Positioned(
                    // The max is needed to also look good on phones with a standard rectangular screen
                    top: max(safePaddingTop + 5, c4),
                    right: c4,
                    child: CustomButton(
                      key: Key("exit_button"),
                      iconData: Icons.clear_rounded,
                      tooltip: 'Exit Poi',
                      onPressed: () {
                        if (hasAR) {
                          unityWidgetController.pause();
                        }
                        if (currentState is TaskScreenFinished) {
                          Navigator.pop(
                              context,
                              NavigatorData(
                                  poi: widget.poi,
                                  completedPOI: true,
                                  returnState: arScreenCubit.taskScreenState,
                                  score: (currentState as TaskScreenFinished).score));
                        } else {
                          Navigator.pop(
                              context,
                              NavigatorData(
                                  poi: widget.poi, completedPOI: false, returnState: arScreenCubit.taskScreenState, score: taskScreenCubit.score));
                        }
                      },
                    ),
                  ),

                  // Submit button only when state is not TaskScreenFinished. So is displayed when answer is (not) selected.
                  Visibility(
                    visible: (!(taskScreenCubit.state is TaskScreenFinished) && modelPlaced),
                    child: Positioned(
                      bottom: 20,
                      right: 20,
                      child: ElevatedButton(
                        key: Key("submit_button"),
                        style: ElevatedButton.styleFrom(
                          shape: RoundedRectangleBorder(
                            borderRadius: BorderRadius.circular(20.0),
                          ),
                          padding: const EdgeInsets.symmetric(horizontal: c4, vertical: c2),
                          onPrimary: cubeGreen,
                          primary: cubeGreen,
                          onSurface: Colors.grey[300],
                        ),
                        child: Text(
                          "Submit",
                          style: CubeFonts.cubeSansSerif(color: Colors.white, fontSize: c3),
                        ),
                        onPressed: (taskScreenCubit.state is TaskScreenAnswerSelected)
                            ? () {
                                TaskScreenAnswerSelected state = taskScreenCubit.state as TaskScreenAnswerSelected;
                                taskScreenCubit.answer = state.answer;
                                taskScreenCubit.submitAnswer();
                                setState(() {
                                  currentState = TaskScreenFinishedModal(
                                      answer: state.answer, score: taskScreenCubit.score, taskData: taskScreenCubit.poi.poiData.task.taskData);
                                });
                                if (widget.poi.poiData.task.taskType == TaskType.DETECTIVE) {
                                  unityWidgetController.postMessage("AR Session Origin", "StopGame", "");
                                }
                                if (widget.poi.poiData.task.taskType == TaskType.GTS) {
                                  unityWidgetController.postMessage("AR Session Origin", "StopGTS",
                                      ((widget.poi.poiData.task.taskData as GuessTheSizeTaskData).size / 100).toString());
                                }
                                _updateGlobalPoiStateOnReturn(context, arScreenCubit, taskScreenCubit);
                              }
                            : null,
                      ),
                    ),
                  ),

                  // Go to Task/Description screen (->) only when state is TaskScreenFinished
                  Visibility(
                    visible: taskScreenCubit.state is TaskScreenFinished,
                    child: Positioned(
                      bottom: 20,
                      right: 20,
                      child: CustomButton(
                        key: Key("task_button"),
                        iconData: Icons.arrow_forward_rounded,
                        tooltip: "Next",
                        onPressed: () {
                          _updateGlobalPoiStateOnReturn(context, arScreenCubit, taskScreenCubit);
                        },
                      ),
                    ),
                  ),

                  // PageIndicator (* *)
                  Padding(
                    padding: const EdgeInsets.only(bottom: c3),
                    child: Align(
                      alignment: Alignment.bottomCenter,
                      child: PageIndicator(
                        onARScreen: true,
                      ),
                    ),
                  ),
                ],
              ),
            );
          },
        ),
      );
    }),
  );
}
